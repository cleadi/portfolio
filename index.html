Semantic Tagging

For this project I added semantic HTML elements in place of div tags to optimize accessibility and SEO. I followed a logical structure independent of styling and positioning and also added accessible alt attributes to HMTL image tags. Heading attributes fall in sequential order. Finally, I adjusted the title to more precisely describe web page.
